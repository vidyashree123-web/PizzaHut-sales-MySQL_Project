# Pizza Hut Sales Analysis Using MySQL

## Project Overview
This project analyzes Pizza Hut sales data using MySQL. The project focuses on sales analysis, revenue generation, customer ordering patterns, and pizza performance using SQL queries.

---

## Objectives
- Total number of orders
- Total revenue analysis
- Most ordered pizza
- Top selling pizza categories
- Order distribution by hour
- Revenue contribution analysis
- Cumulative revenue analysis
